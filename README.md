# 📒 Contact Management System

A simple and user-friendly Contact Management System developed using Java and Java Swing for the graphical user interface.  
This desktop application allows users to manage contacts efficiently with features like adding, updating, deleting, and searching contacts.

---

## 🚀 Features

- ➕ Add new contacts
- ✏️ Update existing contacts
- ❌ Delete contacts
- 🔍 Search contacts
- 📋 View all saved contacts
- 🖥️ Interactive Java Swing GUI
- 💾 Data handling using Java

---

## 🛠️ Technologies Used

- Java
- Java Swing
- OOP Concepts
- IntelliJ IDEA / NetBeans / Eclipse
